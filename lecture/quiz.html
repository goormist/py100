퀴즈 체험하기

문제 풀어보기

수강자는 강의자가 제출한 문제를 보고 하단에 있는 코드창에 문제에 맞는 코드를 작성하면 됩니다.

코드를 작성한 후에 코드창 밑에 '실행' 버튼을 누르면 결과를 확인할 수 있습니다.

'실행' 버튼을 누르면 화면 하단에 터미널 창이 뜨며 터미널창에서 코드 결과를 확인할 수 있습니다.

'찾기' 버튼은 수강자가 작성한 코드 내에서 찾고 싶은 단어나 내용을 검색할 수 있습니다.

문제 제출하기

코드 오른쪽 하단의 제출 버튼을 누르면 정답인지 오답인지 확인할 수 있습니다. 작성한 코드를 기준으로 문제에서 원하는 답인지 아닌지 확인합니다.

1. 총합 구하기

실수 다섯 숫자를 입력 받아 합을 구하시오. (소수점 둘째 자리까지 출력)

예)

input

90.02 88.14 75.64 99.50 45.54

output

398.84

2. 문자열 숫자로 바꾸기

문자를 입력 받아 숫자로 출력하시오.

예)

input

1023

output

1023

3. 최소값 구하기

실수 다섯 개를 입력하고 그 중 최소값를 구하시오.

예)

input

1.5 20.1 16.4 2.3 3.5

output

1.5
