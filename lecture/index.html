구름EDU 강의안 작성법

본 문서는 구름EDU 자습용 강의안을 작성하는 방법에 대한 내용을 설명하는 문서입니다.

구름IDE에서 자습용 프로젝트 만들기

구름IDE에서 구름EDU 자습용 컨텐츠를 만들 수 있습니다. 다시 말해, 구름IDE 프로젝트 = 구름EDU 강의라고 볼 수 있습니다. 따라서, 만약 node.js 강의안을 만들고 싶다면 node.js 프로젝트로 생성되어야 하며, Python 강의안을 만들고 싶다면 Python 프로젝트를 생성해야만 합니다.

구름EDU에서 자습용 프로젝트 실행

구름EDU에서 자습용 프로젝트를 실행하는 방법은 다음과 같습니다.

goorm.io에 로그인이 되어 있다는 전제하에 새창에서,

http://edu.goorm.io/edu/해시태그

로 실행하시면 됩니다. 해시태그는 프로젝트-프로젝트 공유-공유링크에서 다음 밑줄 친 부분입니다.

https://edu.goorm.io/share/ABCDEFGHIJKLMN

강의 목차 만들기

강의 목차는 index.json으로 구성하여 만들 수 있습니다. 각 항목에서 children 값을 이용하여 트리뷰를 계층 구조로 구성할 수 있습니다. 이 때, root 항목 이하 첫번째 깊이의 항목들은 단 하나의 html을 가질 수 있으며, 두번째 깊이의 항목들은 자신의 부모가 가진 html 파일 내부의 h2 항목과 순서대로 연결됩니다.
